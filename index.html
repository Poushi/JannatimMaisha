Research Works

Enhancing Bengali Social Analysis with Multi-class Classification Using Novel Contrastive Learning Techniques and Loss Functions (Extension of Thesis Work)

Dhaka, Bangladesh | April 2024 – Present

Extending thesis work by introducing new loss functions, Token-level Adversarial Contrastive Training (TACT) and Label-aware Contrastive Loss (LCL), to improve sentiment classification in Bengali text. Developed a custom multi-class dataset for text classification. Applied contrastive learning techniques using a pre-trained Bengali BERT model, achieving new benchmarks for binary and multi-class sentiment analysis.

Safeguarding Music Artists Through Machine Learning-Based Singer Classification

Dhaka, Bangladesh | October 2023 – Present | Accepted at AICT 2024

Led a project on identifying and classifying Bengali singers using Convolutional Recurrent Neural Networks (CRNN) and traditional methods. Developed a CRNN model that achieved an F1 score of 0.97, outperforming traditional models in singer classification with Mel-spectrogram features. Created a custom dataset, contributing to reduced copyright infringement and improved music recommendation systems.

Automated Financial Report Detection, Classification, and Structure Recognition Using YOLO and SLANet

Dhaka, Bangladesh | February 2024 – Present | Accepted at FMLDS 2024

Developed a scalable and automated pipeline using YOLO, PP-OCR, and SLANet for financial report extraction. Achieved 99.5% mAP and 98.4% precision in table detection, with 100% accuracy in classification. Reduced processing time to 662.6 ms per page and demonstrated 17.6x faster inference with SLANet. This pipeline automates data extraction, minimizing manual effort and improving analysis efficiency.
